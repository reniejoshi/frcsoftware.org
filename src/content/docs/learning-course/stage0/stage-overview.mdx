Learning the basics of Java is the most important step in learning FRC programming because it builds the foundation.
Using Java, we can write code that allows a robot to run a motor, adjust speed, and much more.
While there is a lot to learn when learning Java, FRCSoftware will only cover the basics needed to program a robot.

Each section will have different activities and exercises to help build up a good understanding of the basics of Java.
If you’re new to programming, it is highly recommended to go through each step in order.
This will help ensure that you don’t miss any important information.
